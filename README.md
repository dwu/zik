ZIK: Zeit-ist-kostbar
=====================

Count-down Clock inspired by [Zeller's Clock](http://www.st.cs.uni-saarland.de/zeller/ZellersClock/).

Based on [Stefan Oskamp's Siemens Wall Clock](http://oskamp.dyndns.org/SiemensClock/SVG/).
